## Open all PDFs in Adobe Acrobat (Windows)

1. Open File Explorer.
2. Go to the project folder:
   - `C:\path\to\focus\projects\quincy-office-remodel\output`
3. Sort by **Type** and select all `.pdf` files.
4. Right-click > **Open with** > **Adobe Acrobat**.
5. In Acrobat, use **Home > Recent** to keep them grouped for batch editing.

## Edit and prepare for accounting/payroll systems

1. In Acrobat, open each PDF and use **Edit PDF** for text/markup updates.
2. Use **Prepare Form** if you need fillable fields (vendor, amount, tax, class, memo).
3. Save edited copies with a suffix, for example:
   - `materials_pricing.acrobat-edit.pdf`
4. Use **Compress PDF** (medium) before upload to reduce transfer issues.

## Share to QuickBooks (recommended path)

QuickBooks does not edit PDFs natively; it stores attachments to transactions.

1. In QuickBooks Online, create or open the target transaction (Estimate/Bill/Expense).
2. Use **Attachments** (paperclip) and upload the edited PDF.
3. For multiple files, zip and attach the zip to a project note if needed.
4. Add a memo naming convention:
   - `Quincy Remodel - <document name> - <YYYY-MM-DD>`

## Share to Square Payroll

Square Payroll typically accepts PDFs as supporting documents for payroll records.

1. In Square Dashboard, open **Payroll**.
2. Go to the related payroll run, team member, or contractor payment record.
3. Upload each edited PDF as a supporting document.
4. Add a note with project and pay-period tags:
   - `Quincy Remodel | Pay Period <start>-<end>`
