- `output/RLCS_Direct_Deposit_Form.pdf`
- `output/RLCS_Contractor_Agreement_Jordan_Welch.pdf`
- `output/RLCS_Contractor_Agreement_Seaira_Sue_Althiede.pdf`
- `output/RLCS_Contractor_Agreement_Lexsys_Dowty.pdf`
- `output/RLCS_Payroll_Onboarding_Package.pdf` (combined packet)

## Adobe Acrobat finalization steps (under thejmillercompany@gmail.com)
1. Sign in to Adobe Acrobat with `thejmillercompany@gmail.com`.
2. Upload all PDFs from this folder into Adobe Document Cloud.
3. For each file, open **Prepare a form** and run auto-detect fields.
4. Convert underlined lines to fillable text fields and checkboxes.
5. Add signature/date fields for employee/contractor/company.
6. Save each finalized file as `*_fillable.pdf`.
7. Open **Combine files** and merge in this order:
   - RLCS_Direct_Deposit_Form_fillable.pdf
   - RLCS_Contractor_Agreement_Jordan_Welch_fillable.pdf
   - RLCS_Contractor_Agreement_Seaira_Sue_Althiede_fillable.pdf
   - RLCS_Contractor_Agreement_Lexsys_Dowty_fillable.pdf
8. Save the combined packet as `RLCS_Payroll_Onboarding_Package_fillable.pdf`.
