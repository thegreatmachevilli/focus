## 3) Adobe Acrobat Step-by-Step (AI Form Builder + Prepare Form)

### Step 1 — Create base documents
1. Open **Adobe Acrobat Pro**.
2. Go to **All tools → Create a PDF**.
3. Build each document from Word/text, or paste content into a DOCX first and convert to PDF.
4. Save files:
   - `RLCS_Direct_Deposit_Form.pdf`
   - `RLCS_Contractor_Agreement_Jordan_Welch.pdf`
   - `RLCS_Contractor_Agreement_Seaira_Sue_Althiede.pdf`
   - `RLCS_Contractor_Agreement_Lexsys_Dowty.pdf`

### Step 2 — Add branding/logo
1. Open each PDF.
2. Use **Edit PDF → Add Image**.
3. Insert logo at top and align consistently.
4. Add header text with company name and document title.

### Step 3 — Use AI-powered form detection
1. Open one PDF.
2. Go to **All tools → Prepare a form**.
3. Choose **Start from this document**.
4. Let Acrobat auto-detect form fields (AI assistance).
5. Review each detected field and rename using the field map above.

### Step 4 — Manually fix field types
For accuracy:
- Routing/account numbers: set as **Text fields** (single line).
- Account type / Yes-No selections: use **radio buttons** or mutually exclusive checkboxes.
- Signatures: add **Signature field** (or text signature line if needed).
- Dates: use date-formatted text fields.
- Long scope text: use **multiline text**.

### Step 5 — Make all agreements fillable
1. Repeat Prepare Form for each contractor agreement.
2. Ensure all key sections are fillable:
   - Contractor information
   - Scope of work
   - Payment terms
   - Signature blocks
3. Use contractor-specific field prefixes to prevent collisions after combining.

### Step 6 — Validate form usability
1. Click **Preview** in Prepare Form.
2. Tab through all fields in order.
3. Confirm no overlapping fields.
4. Check required fields (mark required where appropriate).

### Step 7 — Enable digital signing
1. Add signature fields for both company rep and contractor.
2. Save each PDF.
3. Optional: Use **Request e-signatures** for remote signing workflows.

### Step 8 — Combine into one onboarding package
1. Go to **All tools → Combine files**.
2. Add the four PDFs in this order:
   1) Direct Deposit Form
   2) Jordan Agreement
   3) Seaira Agreement
   4) Lexsys Agreement
3. Click **Combine**.
4. Save as: `RLCS_Payroll_Onboarding_Package.pdf`.

### Step 9 — Final quality check before sending
- Verify logo/header appears on every document.
- Confirm all fields are fillable and tab order is logical.
- Test one full completion + signature cycle.
- Save a locked “template master” copy and a working distribution copy.
